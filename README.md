# Производственный календарь РФ для подписки

Публичный `.ics`-календарь с праздничными, перенесёнными выходными и сокращёнными рабочими днями РФ.

## Подписка

После публикации через GitHub Pages используйте постоянную ссылку:

```text
https://akrenev-droid.github.io/prod-calendar/ru-production-calendar.ics
```

На iPhone: Настройки → Календарь → Учетные записи → Добавить учетную запись → Другое → Подписка на календарь.

## Автообновление

GitHub Actions запускается:

- при каждом push в `main`;
- вручную через `workflow_dispatch`;
- по понедельникам только с июля по октябрь.

Такое окно выбрано по последним постановлениям Правительства РФ о переносе выходных дней:

| Год календаря | Дата постановления |
| --- | --- |
| 2021 | 10 октября 2020 |
| 2022 | 16 сентября 2021 |
| 2023 | 29 августа 2022 |
| 2024 | 10 августа 2023 |
| 2025 | 4 октября 2024 |
| 2026 | 24 сентября 2025 |

Во время запуска workflow:

1. пытается найти и прочитать официальную страницу Правительства РФ о переносе выходных дней на текущий и следующие годы;
2. обновляет `ru_prod_calendar_data.json`, если опубликовано новое постановление;
3. пересобирает `ru-production-calendar.ics`;
4. проверяет структуру `.ics`;
5. публикует файл через GitHub Pages.

Если официальный сайт временно недоступен, workflow оставляет последние подтверждённые данные и всё равно публикует валидный календарь.

## Локальная генерация

```bash
python3 generate_ru_prod_calendar.py --update
python3 validate_ics.py
```
